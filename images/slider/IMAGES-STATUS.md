## 🔄 Próximos Pasos

### Para Personalizar Cada Slide:

1. **Reemplazar imágenes individuales**:
   - `slide-1.jpg` → Imagen de alimentos naturales
   - `slide-2.jpg` → Imagen de bebidas saludables
   - `slide-3.jpg` → Imagen de productos orgánicos

2. **Mantener especificaciones**:
   - Proporción panorámica (2:1 o 3:1)
   - Peso: < 300KB
   - Dimensiones: mínimo 1200px de ancho

3. **Estilo consistente**:
   - Producto destacado
   - Fondo limpio
   - Buena resolución
