# Imágenes del Slider Hero

## Imágenes Requeridas

Crea 3 imágenes para el slider con las siguientes especificaciones:

### Slide 1: Alimentos Naturales
- **Nombre**: `slide-1.jpg`
- **Dimensiones**: 1200x800px (proporción 3:2)
- **Contenido**: Productos de alimentos naturales (cereales, snacks, etc.)
- **Estilo**: Fondo claro (beige/crema), producto centrado
- **Referencia**: Similar al estilo mostrado con lata de Mike's Hard Lemonade

### Slide 2: Bebidas Saludables
- **Nombre**: `slide-2.jpg`
- **Dimensiones**: 1200x800px (proporción 3:2)
- **Contenido**: Bebidas refrescantes (aguas, tés, etc.)
- **Estilo**: Fondo azul claro, producto centrado
- **Elementos**: Puede incluir frutas o elementos decorativos

### Slide 3: Productos Orgánicos
- **Nombre**: `slide-3.jpg`
- **Dimensiones**: 1200x800px (proporción 3:2)
- **Contenido**: Productos orgánicos certificados
- **Estilo**: Fondo verde claro, producto centrado
- **Elementos**: Hojas o elementos naturales

## Recomendaciones de Diseño

1. **Fondo Limpio**: Usar fondos sólidos o gradientes sutiles
2. **Producto Destacado**: El producto debe ser el protagonista
3. **Alta Resolución**: Mínimo 1200px de ancho para verse bien en pantallas grandes
4. **Optimización**: Comprimir las imágenes (< 300KB cada una)
5. **Formato**: JPG o WebP para mejor rendimiento

## Estilo Visual

Basarse en el diseño de referencia:
- Fondo de color sólido con gradiente sutil
- Producto en el centro o lado derecho
- Elementos decorativos minimalistas (frutas, hojas)
- Sombras suaves para dar profundidad
- Colores acordes al branding de Excelon

## Placeholders Temporales

Mientras consigues las imágenes reales, puedes:

1. Usar colores sólidos en CSS (ya configurados)
2. Usar imágenes de ejemplo de servicios como:
   - Unsplash (https://unsplash.com)
   - Pexels (https://pexels.com)
3. Crear mockups simples con Canva o Figma

## Ubicación

Coloca las imágenes en:
```
/images/slider/
├── slide-1.jpg
├── slide-2.jpg
└── slide-3.jpg
```

## Nota

El slider funcionará perfectamente sin imágenes por ahora, mostrando los fondos de colores y textos. Las imágenes mejorarán el impacto visual cuando estén listas.
